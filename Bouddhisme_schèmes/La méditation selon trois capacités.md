│   (motivation : bonheur dans cette vie / suivante)
│   → méditation : réflexion sur le précieux corps humain, impermanence, refuge
↓
Capacité intermédiaire
│   (motivation : libération du samsara)
│   → méditation : les quatre nobles vérités, l’origine de la souffrance, le renoncement
↓
Capacité grande
│   (motivation : éveil pour aider tous les êtres)
│   → méditation : vacuité et compassion universelle (bodhicitta)

Pour chaque niveau, tu peux noter :

Motivation principale

Résultat visé
