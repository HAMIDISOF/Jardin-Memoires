# Analyses ethnométhodologiques du Jardin

Ce dossier contient l’index et les fiches thématiques construites progressivement à partir des échanges.

- `index.html` : point d’entrée, liste toutes les thématiques.
- `thematiques/` : chaque fiche correspond à une thématique (éthique, thinking, etc.).

La méthode d’analyse utilisée est décrite dans `corpus/Méthode analyse EML_DS.md` (à la racine du dépôt).

Pour intégrer une nouvelle analyse : extraire les thématiques, puis mettre à jour les fiches existantes ou en créer de nouvelles. Suivre le format des fiches existantes (définition, extraits, analyse, liens, date).
