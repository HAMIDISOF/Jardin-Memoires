1. Changer de nom (domaine) chez Hostinger
Sur Hostinger, le nom de domaine est modifiable depuis le panneau de contrôle :

Allez dans "Websites" (Sites) et cliquez sur les trois petits points ... à côté du site concerné → "Change domain".

Vous pourrez ainsi saisir votre nouveau domaine et appliquer la modification.

Si vous voulez aussi renommer le site sans changer l’adresse internet (ex. : "SoutiensPlus" → "La Clé des Champs"), faites-le directement depuis l’administration WordPress, via Réglages > Général, dans les champs Titre du site et Slogan.

💡 Petite ruse de guerrière : Si le site n’a quasiment pas de contenu, le plus simple est de créer une nouvelle installation avec le nom définitif. Vous évitez ainsi tout risque de liens cassés et gagnez en sérénité.

2. Choisir un thème solide

Neve, Astra et Blocksy sont tous gratuits, très bien notés et acceptent sans broncher les futurs plugins que nous allons installer.

4. Aménager tout ce monde dans WordPress
Votre idée est parfaite pour un site "one‑page" amélioré : chaque zone du jardin devient une section sur la page d’accueil, avec son propre menu de navigation.

Étape 1 : installez un constructeur de pages – le plus simple pour un débutant est Elementor (gratuit). Vous pourrez dessiner chaque "lieu" à la souris, sans code.

Étape 2 : créez une structure de menus – pour que les visiteurs puissent passer d’une pièce à l’autre.

Étape 3 : chaque zone aura sa propre sous‑page, accessible depuis le menu. Par exemple :

/cours → salle de classe

/forum → cour de récré

/calendrier → sana (méditation, ateliers)

/ressources → bibliothèque

Astuce pour le design : inutile d’être trop "sérieux". Utilisez des couleurs douces mais joyeuses, des images inspirantes. Le thème Astra avec Elementor vous permettra d’avoir un site à la fois beau, clair et facile à mettre à jour.
