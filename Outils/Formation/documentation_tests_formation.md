## 💻 Détails Techniques : Utilisation de Discord (pour les débutants)

### **1. Créer un Serveur**
1. Télécharger Discord : [lien officiel](https://discord.com/download)
2. Créer un compte (email + mot de passe).
3. Cliquer sur **+** (à gauche) → "Créer un serveur".
4. Choisir "Pour moi et mes amis" → Nommer le serveur (ex. "Jardin Coopératif").

### **2. Créer un Salon Textuel**
1. Dans le serveur, cliquer sur **+** à côté de "Salons textuels".
2. Nommer le salon (ex. "#réunion-virtuelle").
3. Inviter les participants via le bouton "Inviter des membres" (lien ou email).

### **3. Utiliser un Bot de Gestion de Parole (ex. 🥕)**
1. Ajouter un bot comme "Dyno" ou "Carl-bot" via [top.gg](https://top.gg/).
2. Configurer les commandes pour gérer la parole (ex. `!jeton @utilisateur`).
3. Tester en équipe : seul·e celui/celle avec le jeton peut parler.

### **4. Envoyer les Invitations**
1. Cliquer sur le nom du serveur → "Inviter des membres".
2. Copier le lien d’invitation et l’envoyer par email/Slack.
3. **Exemple de message** :
   ```
   Salut l’équipe !
   Voici le lien pour la réunion virtuelle de demain : [lien Discord].
   On teste à 10h, prévoir 20 min. Miaou gère la parole avec le jeton 🥕 !
   ```
