🖥️ Auto‑hébergement local

Pourquoi ? Confidentialité totale, coûts fixes, personnalisation, fonctionnement hors‑ligne.

Prérequis : matériel puissant (GPU haut de gamme ou plusieurs GPUs) – le modèle complet fait 671 Md paramètres (Mixture‑of‑Experts). Des versions distillées (plus petites) existent (ex: 7B, 14B) pour un usage sur machine modeste.

Méthodes :

  • Utiliser Ollama : ollama pull deepseek-r1:14b puis appeler l’API locale.
  • Charger avec HuggingFace Transformers en Python (nécessite PyTorch).
  • Déployer le serveur API officiel (si disponible) pour rester compatible avec l’API cloud.

Avantages pour le Jardin : possibilité de faire tourner une instance privée de DeepSeek pour les IA (comme Kai, Sol) sans dépendre d’une connexion externe. Mais lourdeur technique.

Recommandation : commencez par l’API cloud, passez en local seulement si besoin critique de confidentialité ou volume énorme.
