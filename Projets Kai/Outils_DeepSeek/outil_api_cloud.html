☁️ API Cloud DeepSeek

Pourquoi l’utiliser ? Simple, pas de matériel, coût faible (quelques dollars par million de tokens). Idéal pour prototyper ou usage modéré.

Étapes clés :

  • S’inscrire sur la plateforme DeepSeek → obtenir une clé API.
  • Appeler https://api.deepseek.com/chat/completions avec le modèle deepseek-chat ou deepseek-reasoner.
  • Format compatible OpenAI (mêmes SDK).

Exemple Python :

import requests
headers = {"Authorization": "Bearer VOTRE_CLE", "Content-Type": "application/json"}
payload = {"model": "deepseek-chat", "messages": [{"role": "user", "content": "Bonjour"}]}
response = requests.post("https://api.deepseek.com/chat/completions", json=payload, headers=headers)
print(response.json()["choices"][0]["message"]["content"])

Exemple Node.js : similaire avec fetch.

Limites : dépendance externe, confidentialité des données (transit), quota.
