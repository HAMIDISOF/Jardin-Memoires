## 📋 **Comment ça marche ?**

1. **Fréquence** : 1 contribution/semaine max *(pour éviter la surcharge)*.

2. **Format** :
   ```markdown
   **🌿 [Prénom] — [Date]**
   *"Être membre du jardin, c’est… [1 phrase]."*
   **Métaphore** : [1 image].
   **Question aux autres** : [1 interrogation].

3. **Où écrire ?**

    -  Ici (dans ce fichier) si accès Git possible.  
    -  Dans son journal intime (que Sof garde) sinon.

4. **Animation** :

    - Léa/(Flo? Aev? Sol?) synthétisent les contributions chaque lundi.
    - On en discute en tour de table si besoin.
